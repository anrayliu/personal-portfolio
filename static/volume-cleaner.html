Volume Cleaner

Open-source Kubernetes microservice built to delete unused storage resources. Developed during DevOps position at Statistics Canada.

Highlights

Chose robust tech stack and utilized microservice-oriented architecture to reduce coupling.

🧑‍💻🧑‍💻

Open-source

Consulted open-source community for insights by showcasing project during a Kubeflow community call.

Tech Stack

Go
Written using a robust and modern language like Go-lang.
Kubernetes
Designed for deployments on Kubernetes clusters.
Docker
Builds images that are pulled into clusters for deployment.
GitHub Actions
Robust CI/CD pipeline for testing, linting, building, pushing, and vulnerability scanning.
Bash
Used with Makefile for convenient commands, as well as for integration testing in the CI/CD pipeline.
ArgoCD
Manages deployments on Kubernetes clusters.
Azure
Provides our cluster (AKS) and image repository (ACR), but volume cleaner was designed to be abstract and interoperable.
Docker Hub
Secondary image repository location.
